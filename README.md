🚀 Project Overview

The dashboard helps understand customer behavior by analyzing:

Customer subscription status
Revenue by product category
Sales trends by category
Customer age group analysis
Average purchase amount
Customer review ratings
Shipping type preferences
🛠️ Technologies Used
Python (Pandas) – Data Cleaning & Preprocessing
SQL – Data Querying & Analysis
Microsoft Power BI – Data Visualization & Dashboard Creation
📌 Key Features

✔ Interactive dashboard filters
✔ KPI cards for customer insights
✔ Revenue & sales analysis
✔ Customer segmentation analysis
✔ User-friendly and clean dashboard design
